Prow Presubmit for KUnit Gerrit

We, KUnit, use Prow, a Kubernetes based CI/CD system, for our Presubmit system.

Prow has built in support for Gerrit and GitHub integration. The only thing needed to run KUnit on Prow is a special test runner container, which we maintain here: https://kunit.googlesource.com/prow-presubmit.

The Prow config which specifies our container can be found here in the google-oss prow cluster.

For more information, please refer to our Prow container repo: https://kunit.googlesource.com/prow-presubmit/
